My Project
---
 
Welcome to findswim.com

This project aims to help open water swimmers find competitions around the world easily. 

 
 
Setup
---
 
```
npm install
```
 
Usage
---
 
Start the development server with this command:
 
```
npm start
```
 
 
Build
---
 
```
npm run build
```

Deploy on Firebase
---
 
 
```
sudo firebase login
sudo firebase deploy
```
